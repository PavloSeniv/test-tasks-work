## Run Locally

Clone the project

```bash
  git clone https://github.com/PavloSeniv/gulp-build-2022.git
```

Go to the project directory

```bash
  cd gulp-build-2022
```

Install dependencies

```bash
  npm install
```

Start the gulp developer

```bash
  npm run dev
```

Start the gulp build

```bash
  npm run build
```

Start the gulp zip

```bash
  npm run zip
```

Start the gulp ftp

```bash
  npm run ftp
```
